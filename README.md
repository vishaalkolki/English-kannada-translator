# English to Kannada Translator

A Streamlit application that translates English text to Kannada using TranslateGemma running locally with Ollama.

## Features

- English → Kannada translation
- Runs locally using Ollama
- Built with LangChain and Streamlit

## Technologies

- Python
- Streamlit
- LangChain
- Ollama
- TranslateGemma

## Installation

```bash
pip install -r requirements.txt
ollama pull translategemma:4b
streamlit run app.py
```

## Example query: 
How is the weather in Bengaluru today and what are the famous food items I can try there?
